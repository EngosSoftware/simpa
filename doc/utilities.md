# Utilities

---

Utilities are global functions that simplify implementing repeatable tasks.

<div class="alfa">

- [$htmlDiv](/utilities.html#$htmlDiv) Returns an element with specified identifier casted to HTMLDivElement
- [$id](/utilities.html#$id) Returns an element with specified identifier
- [$uuid](/utilities.html#$uuid) Generates a unique identifier (UUID)

</div>

###### <a name="$htmlDiv">$htmlDiv</a>

###### <a name="$id">$id</a>

Returns the element with specified identifier or **null** when element with such an identifier
cannot be found in the DOM tree.

If there is a DIV element somewhere in the DOM tree, like:

```
<div id="my-div">This is my div</div>
```
then calling:

```
const e = $id('my-div');
```

is a shorter form of calling:

```
const e = document.getElementById('my-div');
```

###### <a name="$uuid">$uuid</a>

**$uuid** generates and returns a new UUID identifier. For example, calling:

```
const id = $uuid();
``` 

will return a string containing UUID, like **46c49b9a-ade8-4813-8a32-40f20f2edd56**.
Naturally, each call to **$uuid** will return another unique value.
Identifiers generated by **$uuid** may be used as unique HTML elements' identifiers.
